📰 News Cloud
A sleek, high-performance News App built with Flutter. Focused on speed, security, and a seamless user experience using modern architectural patterns.

🚀 Core Features
Real-time API Integration: Powered by NewsAPI for instant global updates.

In-App Reading: Smooth WebView integration for full article access.

Sliver Architecture: Optimized for fluid scrolling and high performance.

Secure Environment: Sensitive keys are protected via Environment Variables.

🛠 Tech Stack
Network: Dio

Security: Flutter Dotenv

Web Content: WebView Flutter

UI Components: Custom Sliver Widgets

⚙️ Quick Setup
Run flutter pub get.

Create a .env file in the root directory.

Add: NEWS_API_KEY=your_key_here

Run flutter run.
